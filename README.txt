# Screenshot Organizer

This is a simple Python script to organize your gaming or modding screenshots.

## Features
- Moves all .png, .jpg, and .jpeg files from a folder into an "Organized" subfolder.
- Renames files in a numbered sequence: screenshot_001.png, screenshot_002.jpg, etc.

## How to Use
1. Install Python 3 if you don't have it: https://www.python.org/downloads/
